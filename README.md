## 🇬🇧 English Version

Welcome to the central repository of my custom indicators and modifications for the **ATAS** trading platform. Here you will find Order Flow, Volume, and Structure tools optimized for professional trading.

### 🌳 Repository Structure & Branches

To keep development and documentation organized, the repository works as follows:

* **🚀 `publish` Branch (You are here):** This is the **main branch for users**. Here you will find the final documentation, images, and organized indicators ready for consultation.
* **🛠️ Other Branches (`develop`, `feature/*`):** This is where the code lives.
    * **Source Code:** Raw C# files.
    * **Modifications:** Ongoing changes and optimizations.
    * **Pull Requests (PRs):** Improvements I propose to the [official ATAS repository](https://github.com/ATAS-Indicators/ATAS-Indicators).

### 📂 Folder Organization (Publish Branch)

All educational and visual content is organized in the `docs` folder:

* **`📄 /indicators`**: Detailed documentation and technical sheets for each indicator.
    * [View Global Indicator Index (English)](docs/INDICATORS.en.md) *(Link to your EN index)*
* **`🖼️ /img`**: Screenshots and visual diagrams used in the docs.
* **`🎥 /videos`**: Archive of weekly breakdown videos.

### 🌳 Estructura del Repositorio y Ramas

Para mantener el orden entre el desarrollo y la documentación, el repositorio funciona de la siguiente manera:

* **🚀 Rama `publish` (Estás aquí):** Esta es la rama **principal para usuarios y traders**. Aquí encontrarás la documentación final, las imágenes y la estructura organizada de los indicadores listos para consultar.
